# Cafe POS 만들기
에스프레소 종류와 라떼 종류만 파는 카페 POS기의 API를 만들어보자.
----

# 요구 조건

- 카페의 메뉴는 에스프레소와 라떼로 분류된다.
    - 에스프레소는 에스프레소가 들어간 모든 음료.
    - 라떼는 에스프레소를 제외하고 우유가 들어간 모든 음료.
- 소비자가 고른 메뉴는 장바구니에 들어가야 한다.
    - 소비자가 고른 메뉴 가격의 총 합을 도출해야한다.
    - 소비자가 고른 메뉴를 취소할 수 있다.
